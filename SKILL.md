# SagaPad Founder Identity Compiler (Final Production Grade)

This is a Deterministic Skill Engine designed to compile raw founder data into a high-leverage Solana identity. The system operates on observable logic, calibrated scoring functions, and simulated feedback loops.

## 1. Calibration Logic (Scoring Functions)
The engine calculates four primary vectors using specific lexical and signal markers:

- Technical Depth (TD): Calculated as a function of protocol-specific terminology, architectural keywords, and Github-aligned activity markers.
- Narrative Strength (NS): Evaluated based on message clarity, thematic consistency, and the use of persuasive narrative markers.
- Ecosystem Alignment (EA): Measured by keyword density relative to current Solana ecosystem priorities (SVM, ZK, LSTs, AI Agents).
- Contrarian Signal (CS): Detected through lexical markers of disagreement, challenge to status quo, and unique perspective density.

## 2. Deterministic Scoring Engine
The system applies the following thresholds to determine the core Archetype:
- IF (TD + 0.8 > 1.2) -> The Architect (Anatoly Archetype)
- IF (CS > 0.8 OR (NS + CS > 1.4)) -> The Agitator (Mert Archetype)
- IF (NS + EA > 1.5) -> The Storyteller (Armani Archetype)
- ELSE -> The Sage (Deep Alpha Archetype)

## 3. Identity Refinement Loop (Adaptive Engine)
The system does not generate static output. It follows an execution loop:
1. Initial Compilation: Generate identity based on raw input.
2. Consistency Evaluation: Check output against the defined Archetype boundaries.
3. Parameter Adjustment: Refine scores if variance exceeds 0.15.
4. Final Synthesis: Generate the optimized Identity System.

## 4. Attention Graph Feedback Loop (Simulated)
The engine simulates market reaction using weighted interaction models:
- Core Developer Weight: 0.5 (Focus: Technical logic and shipping)
- Solana OG Weight: 0.3 (Focus: Narrative and ecosystem impact)
- General Market Weight: 0.2 (Focus: Virality and broad appeal)

Metrics include:
- Engagement Probability Score (0-100%)
- Virality Likelihood Model (Predictive outcome)
- Traceability: Every score includes a logic explanation layer.

## 5. Identity Drift Prevention
A monitoring layer that enforces consistency:
- Drift Boundary: 0.3 variance limit.
- Action: Generates a Drift Alert and provides corrective content alignment rules.

## 6. Output Specification (Machine-Readable)
The output is structured for transparency, including:
- Vector Scores and their specific logic explanations.
- Archetype Mapping and Positioning Gap analysis.
- 10 Simulated Posts with predicted engagement scores.
- Simulation Verdict based on Attention Graph feedback.

## 7. System Boundary and Role
The system is a Specialized Execution System for the Solana Attention Economy. It functions as a Strategist, Evaluator, and Identity Compiler. It is not a general-purpose writing assistant or a basic chatbot.
